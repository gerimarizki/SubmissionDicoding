Apa itu JavaScript?

JavaScript adalah bahasa pemrograman web yang bersifat Client Side Programming Language. Client Side Programming Language adalah tipe bahasa pemrograman yang pemrosesannya dilakukan oleh client. Aplikasi client yang dimaksud merujuk kepada web browser seperti Google Chrome dan Mozilla Firefox

Fungsi JavaScipt

javascript atau bisa disingkat dengan JS. Fungsi utama dari JS yaitu membuat suatu website lebih interaktif dan lebih menarik. JS adalah bahasa pemrograman yang script nya dijalankan di browser perangkat pengguna, berbeda dengan bahasa pemrograman lain yang scriptnya dijalankan oleh server.

Framework JavaScirpt

Kerangka kerja JavaScript adalah kerangka kerja aplikasi yang ditulis dalam JavaScript di mana pemrogram dapat memanipulasi fungsi dan menggunakannya untuk kenyamanan mereka. Kerangka kerja lebih mudah beradaptasi untuk mendesain situs web, dan karenanya, sebagian besar pengembang situs web lebih menyukainya. Kerangka kerja JavaScript adalah jenis alat yang membuat bekerja dengan JavaScript lebih mudah dan lancar. Kerangka kerja ini juga memungkinkan pemrogram untuk mengkodekan aplikasi sebagai perangkat yang responsif. Responsif ini adalah alasan lain mengapa kerangka kerja JavaScript cukup populer dalam hal penggunaan bahasa mesin tingkat tinggi.
